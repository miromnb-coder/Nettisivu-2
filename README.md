# NordClean – moderni siivouspalvelun nettisivu

Tämä on valmis HTML + CSS + JavaScript -demo modernille siivouspalvelulle.

## Sisältö

- Responsiivinen desktop- ja mobiiliversio
- Hero-osio
- Palvelukortit
- Yrityksille / laatulupaus -osio
- Hinnat-osio
- Arvostelukarouselli
- Yhteystiedot
- Footer ja uutiskirje
- Toimivat modaalit:
  - Tarjouspyyntö
  - Palvelun lisätiedot
  - Meistä
  - FAQ
  - Evästekäytäntö

## Tiedostot

```text
index.html
style.css
script.js
README.md
```

## Julkaisu GitHub Pagesilla

1. Avaa repositoryn Settings
2. Mene Pages-osioon
3. Valitse Source: Deploy from a branch
4. Branch: main
5. Folder: /(root)
6. Tallenna
